## 📍 如何使用？

### 安装

#### 内贸站

```bash | pure
npm i @abiz/rc-jxc
## or
yarn add @abiz/rc-jxc
```

#### 大优采

```bash | pure
npm i @abiz/rc-aeps
## or
yarn add @abiz/rc-aeps
```

#### 小优采

```bash | pure
npm i @abiz/rc-jxc
## or
yarn add @abiz/rc-jxc
```

### 使用

使用方法与`antd`中使用组件的方式一致
