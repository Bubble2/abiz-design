

语义化的矢量图形。使用图标组件，你需要安装 `@ant-design/icons` 图标组件包：

```bash | pure

npm install --save @ant-design/icons
```

## 设计师专属

安装 [Kitchen Sketch 插件 💎](https://kitchen.alipay.com)，就可以一键拖拽使用 Ant Design 和 Iconfont 的海量图标，还可以关联自有项目。
